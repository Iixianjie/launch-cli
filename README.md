# launch-cli
配置模板存放在 <https://github.com/Iixianjie/webpack-templates> 仓库， 切换branche可以查看不同的配置。

## Installation
```
npm install @lxjx/launch-cli -g
or
yarn global add @lxjx/launch-cli  // 使用yarn安装全局前，请先确保你已经正确设置bin目录。
```

<br>

## Usage
### list
查看可用的配置列表
```
launch list
```

<br>

### create
选择模板并创建项目文件
```
launch create <template-name> <project-name>
```

<br>

### -v
查看版本
```
launch -v
```

<br>

### -h
帮助信息
```
launch -h
```
